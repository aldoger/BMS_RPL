Laboratorium Rekayasa Perangkat Lunak

Lab Software Engineering atau Rekayasa Perangkat Lunak adalah lab yang membahas pada bidang software seperti software development, software architecture, web development
Mata kuliah bidang keahlian RPL, antara lain:

  • Penjaminan Mutu Perangkat Lunak,
  • Arsitektur Perangkat Lunak,
  • Evolusi Perangkat Lunak,
  • Konstruksi Perangkat Lunak,
  • Penyempurnaan Proses Perangkat Lunak,
  • Ekonomi Rekayasa Perangkat Lunak,
  • Topik Khusus RPL.
